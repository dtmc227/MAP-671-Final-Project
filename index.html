Finding an arch with a view of the April 8, 2024 solar eclipse

Map by Donovan McGauley

About this project

  • This is a Mapbox style that I created after I used QGIS geoprocessing tools on data from the National File on Geographic Names Information System at https://www.usgs.gov/us-board-on-geographic-names/download-gnis-data and NASA's Scientific Visualization Studio at https://svs.gsfc.nasa.gov/5073. The purpose of this project was to show which United States arches would have a view of the total solar eclipse on April 8th, 2024. For the data I used and addition information about how I created the map, my repository can be found at https://github.com/dtmc227/MAP-671-Final-Project.
  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
